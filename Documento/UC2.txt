Caso de uso 2
Nombre de caso de uso: Login
Breve descripción: El usuario inicia sesión en el sistema
Prerrequisito: estar registrado
Flujo básico: 1) El usuario selecciona la opción "iniciar sesión"
              2) El sistema muestra en pantalla la ventana para ingresar los datos 
              3) El usuario ingresa su email y contraseña 
              4) El sistema valida los datos
              5) El sistema muestra la pantalla de inicio
              

Flujo alternativo: 3a. El sistema muestra que el email o contraseña utilizados son incorrectos, volver a CU2.2
