Caso de uso 1
Nombre de caso de uso: Registrar usuario
Breve descripción: Registra un nuevo usuario en el sistema
Flujo básico: 1) El actor elige la opción registrar un nuevo usuario 
              2) El sistema muestra en pantalla la ventana para ingresar los datos 
              3) El actor ingresa su email y contraseña 
              4) El actor ingresa su nombre y apellido 
              5) El actor presiona el boton "registrarse" 
              6) El sistema almacena la información en la base de datos

Flujo alternativo: 5a. El sistema muestra que el mail ya ha sido utilizado, volver a CU 1.3
